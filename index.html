System Capabilities

Feature Matrix

Anti-Zombie

Reliability

Differentiates between "slow" and "dead". If API returns None 10 times consecutively (10 mins), the worker self-terminates to allow Master to restart it.

Circuit Breaker

Safety

Prevents API bans. If the API fails 3 times globally, the system enters a "Cool-down" state for 30 minutes before trying again.

Auto-Fix (SD)

Data

Automatically detects 400 Bad Request caused by invalid metric columns, removes them from the config, and retries immediately.

UPSERT & Schema Evo

Integrity

Guarantees zero duplicates. Automatically adapts database tables when new columns are added to the configuration.
